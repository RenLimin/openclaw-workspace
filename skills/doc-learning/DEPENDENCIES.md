# doc-learning Dependencies

## 二进制依赖
- `python3`: macOS 自带
- `pandoc`: 文档格式转换 (可选)
  - 安装: `brew install pandoc`

## Python 包依赖
- `pdfminer.six`: PDF 文本提取
  - 安装: `pip3 install pdfminer.six`

## 支持格式
- Markdown (.md) - 原生支持
- 纯文本 (.txt) - 原生支持
- PDF (.pdf) - 纯文本 PDF (非扫描件)

## 注意事项
- 扫描件 PDF 需要 OCR 服务（不在本技能范围）
- 大型文档建议分段处理
