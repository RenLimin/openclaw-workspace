1. 登录 → OA 主页面
2. 左侧导航栏：`销售合同管理系统` → `合同基本信息管理` → `合同台账（销售）`
3. 合同列表页 → 搜索框（"合同编号"）→ 输入编号 → 回车
4. 点击合同编号 → 进入合同详情页（新标签页）
5. 合同详情页 → `基本信息` → `合同文件归档` → `合同扫描件文件`
6. 悬浮光标至文件 → 显示下载按钮 → 触发下载
7. 返回 OA 主页面 → 左侧导航栏：`销售合同管理系统` → `合同基本信息管理` → `销售合同分项查询`
8. 搜索框（"合同编号"）→ 输入编号 → 回车 → 获取分项记录

## 四、技术要点

1. **动态页面等待**: 使用 `wait_for_vue()` / `wait_for_networkidle()` 等待页面加载

3. **文件下载**: 使用 `page.on("download")` 监听下载事件
4. **新标签页处理**: 使用 `context.wait_for_event("page")` 处理新标签页
