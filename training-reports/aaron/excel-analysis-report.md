## 二、核心数据表结构

### 2.1 签约表 (82列)

| 字段组 | 字段示例 |
|--------|---------|
| 项目信息 | BI履约ID, 最终用户名称, 客户名称, 所属项目 |
| 人员信息 | 责任销售, 责任销售所属团队, 负责人 |
| 合同信息 | 销售合同编号, 合同名称, 合同归档日期, 合同起始日期, 合同结束日期 |
| 交付信息 | 预估交付完成日期, 实际服务/授权开始日期, 实际服务/授权结束日期 |
| 验收信息 | 合同验收条款, 验收时点, 验收方式, 预估验收完成日期 |
| 状态信息 | 项目类型, 项目状态, 履约项优先级, 交付邮件状态 |
| 统计信息 | 项目编号, 统计合同编号, 合同归档年度, 项目验收状态 |

## 三、报告制作方式

### 3.1 数据来源
- **签约数据**: 从 CRM/履约系统导出
- **交付数据**: 从项目管理系统导出
- **异常数据**: 从异常跟踪系统导出
- **交接数据**: 从交接管理系统导出

### 3.2 制作流程
1. 导出各系统原始数据
2. 使用 Excel 透视表进行统计
3. 手动调整格式和样式
4. 添加条件格式和图表
5. 生成最终报告

### 3.3 格式特点
- 多层表头结构
- 合并单元格
- 条件格式高亮
- 数据透视表
- 图表可视化
